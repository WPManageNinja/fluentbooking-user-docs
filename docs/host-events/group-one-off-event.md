# Group One-off Event

FluentBooking enables you to create a One-off event, designed for a single occurrence. With a One-off event, you can organize a group event that is for multiple attendees and also applies to a single occasion.

In this guideline, we'll provide a detailed overview of how to create a Group One-off event and explore the features of this event type.


## Create a Group One-off Event

To create a **Group One-off Event**, navigate to the **Calendars** section in the FluentBookings navbar. In the top right corner, click the **New** button, and then select the **One-off Event** option from the drop-down menu.

![Create a Group one-off Event](/images/host-events/group-one-off-event/Booking-One-off-group-event-1-scaled.webp)

A pop-up will appear from the left side, where you'll need to enter the event name and choose the hosts for the meeting. Select **Group One-off Event** here and you'll be redirected to the event details page for further customization.

![Customization](/images/host-events/group-one-off-event/Booking-One-off-group-event-2-scaled.webp)

The event detail page will appear. Enter the necessary details about your event and click the **Continue** button to proceed.

![Continue Button](/images/host-events/group-one-off-event/Booking-One-off-group-event-3-scaled.webp)


## Manage Hosts

In the Assignment section, you can manage the hosts for your meeting. If needed, you can add more hosts using the **Add Hosts** dropdown field.

In the **Event Hosts** section, you'll see the hosts already added to your event. To designate a host as an **Organizer**, simply click the **Make Organizer** button, then click **Save changes** to apply your updates.

![Manage Hosts](/images/host-events/group-one-off-event/Booking-One-off-group-event-4-scaled.webp)


## Availability

In the **Availability** section of your Group One-off Event, you can define the event's available time slots.

If you enable the Reserve Time option, the host's availability will be exclusively reserved for this event. Once an attendee selects a booking time, such as 9:00 - 9:30, that time slot will be blocked, preventing the host from being booked for any other events during that period.

To set the availability time, click the **Add Available Times** button.

![Availability](/images/host-events/group-one-off-event/Booking-One-off-group-event-5-scaled.webp)

A pop-up will appear for selecting the availability time for the booking. From here, you can choose only one time slot fro your event. Once you've made your selections, click the **Apply** button to save the changes.

![Apply Button](/images/host-events/group-one-off-event/Booking-One-off-group-event-6-scaled.webp)

The remaining settings for the booking event are similar to those of other events.


## Share the Event

To share this booking event, click the **Share** button of the event in the **Calendars** section.

![Share the Event](/images/host-events/group-one-off-event/Booking-One-off-group-event-8-scaled.webp)

A pop-up will appear, allowing you to choose how you want to share the event for attendees to make their bookings.

![Share the Event-1](/images/host-events/group-one-off-event/Booking-One-off-group-event-9-scaled.webp)
